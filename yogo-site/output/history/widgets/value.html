Value widget

In this database system, simple integer values can be entered into a text field widget. The value widget, however, provides a richer mechanism for specifying a number. Specifically, the value widget provides additional menus for specifying units (inches, mm, microns, kilograms, etc) and an optional confidence factor.

Example:


In the Create Form tab, click on the value widget icon in the toolbar.



You will be given the choice of creating a whole number or decimal number widget.



For this example, we chose a decimal number. The widget appears with a default label.



Double click on the label to change it to an appropriate name.



The widget is now ready to record data in Enter Data mode.
